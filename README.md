# Employee Analytics Report (SAP ABAP)

## 📌 Overview

This project is an Employee Analytics Report built using SAP ABAP in Eclipse ADT. It retrieves employee data from a custom database table and performs business logic calculations.

## ⚙️ Features

* Fetch top 50 employees based on salary
* Bonus calculation (10% of salary)
* Employee categorization:

  * Junior (≤2 years)
  * Mid (3–5 years)
  * Senior (>5 years)
* Total employee count
* Total salary calculation

## 🛠️ Technologies Used

* SAP ABAP (Eclipse ADT)
* Open SQL
* Custom Table (ZEMPLOYEE)

## 📊 Output

Displays structured employee report with analytics in console.
